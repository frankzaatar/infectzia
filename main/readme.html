הערת המפתח

ברוכים הבאים. WordPress היא פרוייקט מיוחד מאוד בשבילי. כל המפתחים והתורמים האחרים מוסיפים משהו למערכת, וביחד אנחנו יוצרים משהו יפהפה שאני גאה להיות חלק בו. אלפי שעות הושקעו ב-WordPress, ואנחנו ממשיכים לשפר אותה מדי יום. תודה לך על שהכנסת אותה לעולם שלך.

מאט מולנווג

התקנה - חמש דקות קצרות

ברוב המקרים, ההתקנה של WordPress יכולה להתבצע כולה דרך הדפדפן. אם פעולות כמו עריכת קובצי PHP נשמעות לכם מדאיגות, פשוט נסו להעלות את כל הקבצים של WordPress לשרת ולגשת לכתובת שלה בדפדפן. עם זאת, זכרו שלא תוכלו להתחמק מדברים כאלה לנצח...

  1. פתחו את הקובץ שהורדתם (wordpress.zip או wordpress.tgz) בתיקיה ריקה.
  2. פתחו את הקובץ wp-config-sample.php בעורך טקסט )כמו Notepad, לא כמו Word(, ומלאו את פרטי החיבור לבסיס הנתונים, לפי ההוראות.
  3. שמרו את הקובץ בשם ‎wp-config.php‏.
  4. העלו את הכל לאתר.
  5. הפנו את הדפדפן לכתובת ‎/wp-admin/install.php‏ - תוכנית ההתקנה תתקין את וורדפרס בבסיס הנתונים. אם ההתקנה נכשלת, בדקו את את ההגדרות בקובץ wp-config.php‏. אם ההתקנה עדיין נכשלת, פנו לפורומים עם נתונים רבים ככל האפשר אודות ההתקנה.
  6. רשמו את הסיסמה במקום בטוח, או בכל אופן זכרו אותה בצורה כלשהיא.
  7. לחצו על הקישור לעמוד הכניסה. הכניסו את שם הכניסה 'admin' ואת הסיסמה שקיבלתם בעמוד הקודם.
  8. סיימתם את ההתקנה, ואתם נמצאים עתה בממשק הניהול של הבלוג שלכם. מזל טוב!

שדרוג - שלושה שלבים קלים

לפני השדרוג, הכינו גיבויים של כל הקבצים ששיניתם, כמו קובצי העיצוב או תוספים שהתקנתם, וכן של בסיס נתונים שלכם. וורדפרס 2 צריכה לעשות שינויים בבסיס הנתונים בעת שדרוג מגרסאות קודמות, וכדאי מאוד שיהיה לכם גיבוי של כל הפוסטים והתגובות למקרה שמשהו ישתבש.

  1. צרו גיבוי של כל התיקה של וורדפרס, וגבו גם את בסיס הנתונים.
  2. העלו את וורדפרס 2.0 על גבי ההתקנה הנוכחית של וורדפרס
  3. פתחו את הכתובת ‎ /wp-admin/upgrade.php ‏ בדפדפן.

שינויים בעיצוב

אם עשיתם שינויים בעיצוב הרגיל של וורדפרס, ייתכן שתצטרכו להעלות אותם מחדש לשרת אחרי השדרוג. אם אתם משדרגים מגרסה 1.2 או מגרסה מוקדמת יותר, קראו את ההוראות האלה.
